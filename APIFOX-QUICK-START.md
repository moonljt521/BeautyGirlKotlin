# 🚀 Apifox MCP 快速开始

## 一分钟配置

```bash
# 运行配置脚本
bash scripts/setup-apifox-mcp.sh
```

按提示输入你的 Apifox Access Token，然后重启 Kiro IDE。

## 获取 Access Token

1. 访问 https://apifox.com
2. 登录 → 个人设置 → Access Token
3. 生成新 Token → 复制

## 立即使用

重启 Kiro 后，在聊天窗口中输入：

```
列出我的 Apifox 项目
```

## 常用命令

| 你说 | AI 做什么 |
|------|----------|
| 列出我的项目 | 显示所有 Apifox 项目 |
| 列出所有接口 | 显示项目中的所有 API |
| 调用 Gank API | 测试 Gank API 接口 |
| 测试所有 API | 批量测试所有接口 |
| 生成测试报告 | 生成详细的测试报告 |
